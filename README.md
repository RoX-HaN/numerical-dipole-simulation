# ⚡ Dipole Potential Simulator

A C++ project that numerically computes the electric potential of a dipole and exports the data for visualization.

---

## 📌 Overview

This project simulates the electric potential generated by an electric dipole using fundamental electrostatics equations.
It generates a 2D grid of potential values and stores them in a CSV file for further visualization.

---

## 🧠 Physics Concept

An electric dipole consists of two equal and opposite charges separated by a small distance.
The electric potential at any point is calculated using:

V = kQ (1/r₊ - 1/r₋)

Where:

* `k` = Coulomb's constant
* `Q` = charge magnitude
* `r₊`, `r₋` = distances from positive and negative charges

---

## ⚙️ Features

* 📐 Grid-based numerical simulation
* ⚡ Computes electric potential at each point
* 📁 Exports results to CSV file
* 🧪 Based on real physics equations

---

## 📂 Output

The program generates:

dipole_potential.csv

Each row contains:
x, y, potential

---

## 🚀 How to Run

### 1. Compile

```bash
g++ dipole.cpp -o dipole
```

### 2. Run

```bash
./dipole
```

After running, the CSV file will be generated in the same folder.
